## 🎯 준비 단계

### 1단계: 기초 다지기 (1-2주)
- **알고리즘 기초**: 시간복잡도, 공간복잡도 이해
- **자료구조**: 배열, 문자열, 해시맵, 스택, 큐, 힙, 트리, 그래프
- **기본 문법**: 선택한 언어의 기본 문법 완벽 숙지

### 2단계: 알고리즘 학습 (3-4주)
- **정렬**: 버블, 선택, 삽입, 병합, 퀵, 힙 정렬
- **탐색**: 이진 탐색, DFS, BFS
- **동적 프로그래밍**: 메모이제이션, 탑다운/바텀업
- **그리디**: 최적화 문제 해결

### 3단계: 문제 유형별 연습 (4-6주)
- **구현**: 시뮬레이션, 완전탐색
- **수학**: 소수, 최대공약수, 최소공배수
- **문자열**: 문자열 처리, 정규표현식
- **그래프**: 최단경로, 최소신장트리, 위상정렬

### 4단계: 실전 연습 (2-3주)
- **기업별 코딩테스트**: 카카오, 네이버, 삼성, LG 등
- **온라인 저지**: 백준, 프로그래머스, 리트코드
- **모의고사**: 실제 시험 환경 연습

## 🎮 실전 연습 전략

### 1. 문제 풀이 순서
1. **문제 이해**: 입력/출력, 제약조건 파악
2. **예시 분석**: 손으로 계산해보기
3. **알고리즘 설계**: 접근 방법 결정
4. **코드 작성**: 깔끔하고 효율적으로
5. **테스트**: 예시 케이스 검증
6. **최적화**: 시간/공간 복잡도 개선

### 2. 디버깅 전략
- **출력 디버깅**: 중간 결과 출력
- **단계별 검증**: 각 단계별 결과 확인
- **예외 케이스**: 경계값, 예외 상황 테스트

### 3. 시간 관리
- **문제 읽기**: 5분
- **알고리즘 설계**: 10분
- **코드 작성**: 20분
- **테스트/디버깅**: 15분

## 📅 일정 관리

### 월간 목표
- **1월**: 기초 다지기 + 기본 알고리즘
- **2월**: 문제 유형별 연습
- **3월**: 실전 연습 + 모의고사

### 주간 목표
- **평일**: 하루 2-3문제 풀이
- **주말**: 복습 + 새로운 개념 학습
- **일요일**: 주간 진도 체크

### 일일 목표
- **오전**: 새로운 개념 학습 (1시간)
- **오후**: 문제 풀이 (2시간)
- **저녁**: 복습 및 정리 (1시간)

## 🎯 성공 전략

### 1. 꾸준함
- 매일 조금씩이라도 꾸준히 학습
- 주말에는 더 많은 시간 투자

### 2. 체계적 접근
- 기초부터 차근차근 학습
- 이해가 안 되면 돌아가서 다시 학습

### 3. 실전 중심
- 이론보다 실습 위주로 학습
- 다양한 문제 유형 경험

### 4. 피드백 활용
- 틀린 문제는 반드시 복습
- 다른 사람의 풀이도 참고
