## 🎯 기본 문법의 중요성

코딩테스트에서 기본 문법을 완벽히 숙지하는 것은 필수입니다. 문법 오류로 인한 시간 낭비를 방지하고, 문제 해결에 집중할 수 있도록 해야 합니다.

## 📚 Python 기본 문법 (추천 언어)

### 1. 변수와 데이터 타입
```python
# 숫자
a = 10          # 정수
b = 3.14        # 실수
c = 1e9         # 과학적 표기법 (10^9)

# 문자열
s = "Hello"
s2 = 'World'
s3 = """여러 줄
문자열"""

# 불린
flag = True
flag2 = False

# 리스트 (가변)
arr = [1, 2, 3, 4, 5]
arr.append(6)    # 추가
arr.pop()        # 마지막 요소 제거
arr[0]           # 인덱싱
arr[1:3]         # 슬라이싱

# 튜플 (불변)
tup = (1, 2, 3)
# tup[0] = 4    # 에러! 수정 불가

# 딕셔너리
dict = {'key': 'value', 'name': 'John'}
dict['new_key'] = 'new_value'  # 추가
del dict['key']                 # 삭제

# 집합
set1 = {1, 2, 3, 4}
set1.add(5)     # 추가
set1.remove(1)  # 삭제
```

### 2. 입출력 처리
```python
# 기본 입력
n = int(input())                    # 정수 입력
s = input()                         # 문자열 입력
arr = list(map(int, input().split()))  # 공백으로 구분된 정수 배열

# 빠른 입력 (대용량 데이터)
import sys
input = sys.stdin.readline
n = int(input().rstrip())

# 출력
print("Hello World")                # 기본 출력
print(f"n = {n}")                  # f-string
print("a = {}, b = {}".format(a, b))  # format 메서드
print("a = %d, b = %d" % (a, b))   # % 연산자

# 줄바꿈 없이 출력
print("Hello", end=" ")
print("World")  # Hello World
```

### 3. 조건문과 반복문
```python
# if-elif-else
if x > 0:
    print("양수")
elif x < 0:
    print("음수")
else:
    print("0")

# for 반복문
for i in range(10):        # 0~9
    print(i)

for i in range(1, 11):     # 1~10
    print(i)

for i in range(0, 10, 2):  # 0, 2, 4, 6, 8
    print(i)

# 리스트 순회
arr = [1, 2, 3, 4, 5]
for item in arr:
    print(item)

for i, item in enumerate(arr):
    print(f"인덱스: {i}, 값: {item}")

# while 반복문
i = 0
while i < 10:
    print(i)
    i += 1

# break와 continue
for i in range(10):
    if i == 5:
        break       # 반복문 종료
    if i == 2:
        continue    # 다음 반복으로 건너뛰기
    print(i)
```

### 4. 함수 정의
```python
# 기본 함수
def add(a, b):
    return a + b

# 기본값 매개변수
def greet(name="World"):
    return f"Hello, {name}!"

# 가변 인수
def sum_all(*args):
    return sum(args)

# 키워드 가변 인수
def print_info(**kwargs):
    for key, value in kwargs.items():
        print(f"{key}: {value}")

# 람다 함수
square = lambda x: x ** 2
add = lambda x, y: x + y
```

### 5. 리스트 컴프리헨션
```python
# 기본 형태
squares = [x**2 for x in range(10)]

# 조건 추가
even_squares = [x**2 for x in range(10) if x % 2 == 0]

# 중첩 반복
matrix = [[i+j for j in range(3)] for i in range(3)]

# 딕셔너리 컴프리헨션
square_dict = {x: x**2 for x in range(5)}
```

### 6. 유용한 내장 함수
```python
# 수학 함수
abs(-5)         # 절댓값
max([1, 2, 3]) # 최댓값
min([1, 2, 3]) # 최솟값
sum([1, 2, 3]) # 합계
pow(2, 3)       # 거듭제곱

# 정렬
arr = [3, 1, 4, 1, 5, 9]
sorted_arr = sorted(arr)        # 새 리스트 반환
arr.sort()                      # 원본 변경

# 역순 정렬
sorted_arr = sorted(arr, reverse=True)
arr.sort(reverse=True)

# 문자열 함수
s = "Hello World"
s.upper()       # 대문자 변환
s.lower()       # 소문자 변환
s.split()       # 공백으로 분할
s.replace('o', '0')  # 문자 교체

# 리스트 함수
arr = [1, 2, 3, 4, 5]
len(arr)        # 길이
arr.count(1)    # 특정 값 개수
arr.index(3)    # 특정 값의 인덱스
```


### 2. 자주 사용하는 라이브러리
```python
# collections
from collections import defaultdict, Counter, deque

# itertools
from itertools import permutations, combinations, product

# heapq (우선순위 큐)
import heapq

# math
import math

math.lcm(a, b)      # 최소공배수
math.sqrt(x)        # 제곱근
```

### 3. 2차원 배열 처리
```python
# 2차원 배열 생성
n, m = 3, 4
arr = [[0] * m for _ in range(n)]

# 2차원 배열 순회
for i in range(n):
    for j in range(m):
        print(arr[i][j])

# 전치 (행렬 뒤집기)
transposed = list(zip(*arr))
```

### 4. 문자열 처리
```python
# 문자열 분할
s = "apple,banana,orange"
fruits = s.split(',')

# 문자열 결합
result = ','.join(fruits)

# 문자열 치환
s = "Hello World"
s = s.replace('o', '0')

# 문자열 검색
if 'Hello' in s:
    print("찾음")

# 문자열 슬라이싱

print(s[0:5])    # Hello
```
