# 학습 계획 가이드라인

## 🎯 학습 계획의 중요성

체계적인 학습 계획은 코딩테스트 준비의 성공을 좌우하는 핵심 요소입니다. 개인의 수준과 목표에 맞는 실현 가능한 계획을 수립하고 지속적으로 관리하는 것이 중요합니다.

## 📊 학습 계획 수립 방법

### 1. 현재 상태 진단
**진단 요소:**
- **기본 실력**: 프로그래밍 언어 숙련도
- **알고리즘 지식**: 알고리즘 이해도
- **문제 해결 능력**: 실제 문제 해결 경험
- **시간 투자 가능량**: 일일/주간 학습 시간

**진단 템플릿:**
```python
def current_status_assessment():
    assessment = {
        "programming_skills": {
            "language_proficiency": "언어 숙련도 (1-5)",
            "basic_concepts": "기본 개념 이해도",
```

### 2. 목표 설정
**목표 유형:**
- **단기 목표**: 1-3개월 내 달성 목표
- **중기 목표**: 3-6개월 내 달성 목표
- **장기 목표**: 6개월 이상 달성 목표

**목표 설정 템플릿:**
```python
def goal_setting():
    goals = {
        "short_term": {
            "duration": "1-3개월",

            "targets": ["중급 알고리즘 마스터", "목표 기업 코딩테스트 합격"],
            "metrics": ["모의고사 성과", "기업별 문제 해결 능력"]
        },
        "long_term": {
            "duration": "6개월 이상",
            "targets": ["고급 알고리즘 마스터", "원하는 기업 입사"],

```

### 3. 학습 계획 수립
**계획 구성:**
- **일일 계획**: 하루 단위 학습 목표
- **주간 계획**: 주 단위 학습 진도
- **월간 계획**: 월 단위 학습 성과

**계획 수립 템플릿:**

- 기본 프로그래밍 문법 숙지
- 기본 알고리즘 이해
- 간단한 문제 해결 능력 향상

**학습 계획:**
