# 정렬 알고리즘 학습 가이드라인

## 🎯 정렬 알고리즘의 중요성

정렬은 코딩테스트에서 가장 기본적이면서도 중요한 알고리즘입니다. 많은 문제에서 정렬이 전처리 단계로 사용되며, 정렬 알고리즘의 특성을 이해하면 다양한 문제를 효율적으로 해결할 수 있습니다.

## 📊 정렬 알고리즘 비교표

| 알고리즘 | 평균 시간복잡도 | 최악 시간복잡도 | 공간복잡도 | 안정성 | 특징 |
|---------|---------------|---------------|-----------|--------|------|
| 버블 정렬 | O(n²) | O(n²) | O(1) | 안정 | 구현 간단 |
| 선택 정렬 | O(n²) | O(n²) | O(1) | 불안정 | 교환 횟수 최소 |
| 삽입 정렬 | O(n²) | O(n²) | O(1) | 안정 | 작은 데이터에 효율적 |
| 병합 정렬 | O(n log n) | O(n log n) | O(n) | 안정 | 분할 정복 |
| 퀵 정렬 | O(n log n) | O(n²) | O(log n) | 불안정 | 평균적으로 가장 빠름 |
| 힙 정렬 | O(n log n) | O(n log n) | O(1) | 불안정 | 추가 메모리 불필요 |

## 🔧 기본 정렬 알고리즘

### 1. 버블 정렬 (Bubble Sort)
**개념:** 인접한 두 원소를 비교하여 순서가 잘못되어 있으면 교환

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        # 이미 정렬된 부분은 제외
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
    return arr

# 최적화된 버블 정렬
def optimized_bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        # 교환이 없으면 이미 정렬됨
        if not swapped:
            break
    return arr
```

**특징:**
- 구현이 매우 간단
- 작은 데이터에 적합
- 안정 정렬

### 2. 선택 정렬 (Selection Sort)
**개념:** 최솟값을 찾아서 앞쪽부터 차례대로 배치

```python
def selection_sort(arr):
    n = len(arr)
    for i in range(n):
        # 최솟값의 인덱스 찾기
        min_idx = i
        for j in range(i + 1, n):
            if arr[j] < arr[min_idx]:
                min_idx = j
        # 최솟값을 현재 위치로 이동
        arr[i], arr[min_idx] = arr[min_idx], arr[i]
    return arr
```

**특징:**
- 교환 횟수가 적음
- 불안정 정렬
- 작은 데이터에 적합

### 3. 삽입 정렬 (Insertion Sort)
**개념:** 현재 원소를 앞쪽의 정렬된 부분에 삽입

```python
def insertion_sort(arr):
    for i in range(1, len(arr)):
        key = arr[i]
        j = i - 1
        # key보다 큰 원소들을 뒤로 이동
        while j >= 0 and arr[j] > key:
            arr[j + 1] = arr[j]
            j -= 1
        arr[j + 1] = key
    return arr
```

**특징:**
- 작은 데이터에 매우 효율적
- 거의 정렬된 데이터에 최적
- 안정 정렬

## 🚀 고급 정렬 알고리즘

### 1. 병합 정렬 (Merge Sort)
**개념:** 분할 정복 방식으로 배열을 반으로 나누어 정렬 후 병합

```python
def merge_sort(arr):
    if len(arr) <= 1:
        return arr
    
    # 분할
    mid = len(arr) // 2
    left = merge_sort(arr[:mid])
    right = merge_sort(arr[mid:])
    
    # 병합
    return merge(left, right)

def merge(left, right):
    result = []
    i = j = 0
    
    while i < len(left) and j < len(right):
        if left[i] <= right[j]:
            result.append(left[i])
            i += 1
        else:
            result.append(right[j])
            j += 1
    
    # 남은 원소들 추가
    result.extend(left[i:])
    result.extend(right[j:])
    
    return result

# 제자리 병합 정렬 (공간복잡도 O(1))
def in_place_merge_sort(arr, left=0, right=None):
    if right is None:
        right = len(arr) - 1
    
    if left < right:
        mid = (left + right) // 2
        in_place_merge_sort(arr, left, mid)
        in_place_merge_sort(arr, mid + 1, right)
        merge_in_place(arr, left, mid, right)

def merge_in_place(arr, left, mid, right):
    # 두 부분 배열을 병합
    temp = arr[left:right + 1]
    i = 0
    j = mid - left + 1
    k = left
    
    while i <= mid - left and j <= right - left:
        if temp[i] <= temp[j]:
            arr[k] = temp[i]
            i += 1
        else:
            arr[k] = temp[j]
            j += 1
        k += 1
    
    # 남은 원소들 복사
    while i <= mid - left:
        arr[k] = temp[i]
        i += 1
        k += 1
    
    while j <= right - left:
        arr[k] = temp[j]
        j += 1
        k += 1
```

**특징:**
- 안정 정렬
- 외부 정렬에 사용
- 병렬 처리에 적합

### 2. 퀵 정렬 (Quick Sort)
**개념:** 피벗을 기준으로 작은 값과 큰 값으로 분할하여 정렬

```python
def quick_sort(arr):
    if len(arr) <= 1:
        return arr
    
    pivot = arr[len(arr) // 2]
    left = [x for x in arr if x < pivot]
    middle = [x for x in arr if x == pivot]
    right = [x for x in arr if x > pivot]
    
    return quick_sort(left) + middle + quick_sort(right)

# 제자리 퀵 정렬
def quick_sort_in_place(arr, low=0, high=None):
    if high is None:
        high = len(arr) - 1
    
    if low < high:
        pivot_index = partition(arr, low, high)
        quick_sort_in_place(arr, low, pivot_index - 1)
        quick_sort_in_place(arr, pivot_index + 1, high)

def partition(arr, low, high):
    # 피벗 선택 (마지막 원소)
    pivot = arr[high]
    i = low - 1
    
    for j in range(low, high):
        if arr[j] <= pivot:
            i += 1
            arr[i], arr[j] = arr[j], arr[i]
    
    arr[i + 1], arr[high] = arr[high], arr[i + 1]
    return i + 1

# 개선된 퀵 정렬 (3-way partition)
def quick_sort_3way(arr, low=0, high=None):
    if high is None:
        high = len(arr) - 1
    
    if low < high:
        lt, gt = partition_3way(arr, low, high)
        quick_sort_3way(arr, low, lt - 1)
        quick_sort_3way(arr, gt + 1, high)

def partition_3way(arr, low, high):
    pivot = arr[low]
    lt = low
    gt = high
    i = low + 1
    
    while i <= gt:
        if arr[i] < pivot:
            arr[lt], arr[i] = arr[i], arr[lt]
            lt += 1
            i += 1
        elif arr[i] > pivot:
            arr[gt], arr[i] = arr[i], arr[gt]
            gt -= 1
        else:
            i += 1
    
    return lt, gt
```

**특징:**
- 평균적으로 가장 빠름
- 캐시 효율적
- 불안정 정렬

### 3. 힙 정렬 (Heap Sort)
**개념:** 힙 자료구조를 이용하여 정렬

```python
def heap_sort(arr):
    n = len(arr)
    
    # 최대 힙 구성
    for i in range(n // 2 - 1, -1, -1):
        heapify(arr, n, i)
    
    # 힙에서 하나씩 추출
    for i in range(n - 1, 0, -1):
        arr[0], arr[i] = arr[i], arr[0]
        heapify(arr, i, 0)

def heapify(arr, n, i):
    largest = i
    left = 2 * i + 1
    right = 2 * i + 2
    
    if left < n and arr[left] > arr[largest]:
        largest = left
    
    if right < n and arr[right] > arr[largest]:
        largest = right
    
    if largest != i:
        arr[i], arr[largest] = arr[largest], arr[i]
        heapify(arr, n, largest)
```

**특징:**
- 추가 메모리 불필요
- 우선순위 큐 구현에 사용
- 불안정 정렬

## 🔧 특수한 정렬 알고리즘

### 1. 계수 정렬 (Counting Sort)
**개념:** 각 원소의 개수를 세어서 정렬

```python
def counting_sort(arr):
    if not arr:
        return arr
    
    # 최댓값 찾기
    max_val = max(arr)
    
    # 카운트 배열 생성
    count = [0] * (max_val + 1)
    
    # 각 원소의 개수 세기
    for num in arr:
        count[num] += 1
    
    # 누적 개수 계산
    for i in range(1, len(count)):
        count[i] += count[i - 1]
    
    # 결과 배열 생성
    result = [0] * len(arr)
    for num in reversed(arr):
        result[count[num] - 1] = num
        count[num] -= 1
    
    return result
```

**특징:**
- 정수 데이터에만 사용 가능
- 범위가 작을 때 매우 효율적
- 안정 정렬

### 2. 기수 정렬 (Radix Sort)
**개념:** 각 자릿수별로 정렬

```python
def radix_sort(arr):
    if not arr:
        return arr
    
    # 최댓값의 자릿수 찾기
    max_val = max(arr)
    exp = 1
    
    while max_val // exp > 0:
        counting_sort_by_digit(arr, exp)
        exp *= 10

def counting_sort_by_digit(arr, exp):
    n = len(arr)
    output = [0] * n
    count = [0] * 10
    
    # 각 자릿수의 개수 세기
    for i in range(n):
        index = (arr[i] // exp) % 10
        count[index] += 1
    
    # 누적 개수 계산
    for i in range(1, 10):
        count[i] += count[i - 1]
    
    # 결과 배열 생성
    for i in range(n - 1, -1, -1):
        index = (arr[i] // exp) % 10
        output[count[index] - 1] = arr[i]
        count[index] -= 1
    
    # 원본 배열에 복사
    for i in range(n):
        arr[i] = output[i]
```

**특징:**
- 정수 데이터에만 사용 가능
- 자릿수가 적을 때 효율적
- 안정 정렬

## 📊 성능 비교

### 시간복잡도별 특징
- **O(n²)**: 작은 데이터 (n < 50)
- **O(n log n)**: 일반적인 데이터
- **O(n)**: 특수한 경우 (계수 정렬, 기수 정렬)

### 메모리 사용량
- **제자리 정렬**: O(1) - 버블, 선택, 삽입, 힙
- **비제자리 정렬**: O(n) - 병합, 퀵
